## 기능 요약 + 기존 코드 연결점/의존성 (에이전트용)
1. **목표 달성 시스템** — 목표(월/연 배당·FIRE·은퇴시점) + 달성률·증감·예상 달성일 + SNS 공유 카드.
   - 연결: `sim_summary`에 이미 `targetMonthlyDividend`/`targetReachedInYears` 저장됨. OG 카드(`api/og.tsx`) 인프라 재사용. 진행 중인 "카드 정보 확장"의 연장선.
   - 신규: "이번 달 증가량/지난달 대비"는 시계열 스냅샷 저장이 필요(현재는 단일 시점) → 상태·DB 확장.
2. **ETF 비교** — SCHD vs VIG 등, 배당률/CAGR/총수익/배당성장/지급월/운용보수/최대낙폭/섹터/안정성.
   - ⚠️ **데이터 의존성 최대**: 현재 티커 데이터(가격·배당률·주기·expectedTotalReturn)로는 부족. 운용보수·섹터·최대낙폭·정식 배당히스토리 = 신규 데이터 소스 필요(**FMP 무료 불가 확인됨, 402** — architecture.md 토대 B 참고) → ticker-data-curator + 파이프라인. SEO 핵심.
3. **FIRE Dashboard** — 현재 포트폴리오 그대로 사용. 목표 생활비·국민연금·기타수입 입력 → 커버율·예상 FIRE. 생활비 항목별(식비/주거/통신) 배당 커버 시각화.
   - 연결: 시뮬 엔진 결과(세후 월배당) 재사용. 신규 입력값 → 폼·상태 확장.
4. **배당 캘린더** — 월별 배당락일·지급일·예상 배당금·총 입금액. 재방문 유도.
   - ⚠️ 데이터: 티커별 배당 지급 스케줄(배당락/지급일) 필요 — 현재 미보유.
5. **ETF 상세 페이지** (`/etf/SCHD`) — SEO. 배당률·운용보수·섹터·성장률·지급월·배당히스토리·유사 ETF. #2와 데이터 공유.
6. **실제 배당 기록** — 예상이 아닌 실수령 배당 입력·기록. 월/연 그래프·누적·목표 대비. 리텐션. 신규 DB 모델.
7. **포트폴리오 복사 + 내 조건으로 계산** — 커뮤니티 시나리오 → "내 조건으로 계산" → 저장. **핵심 바이럴.**
   - 연결: 이미 있는 "시뮬레이터로 열기"(`useScenarioDetail` openInSimulatorHref) + 클라우드 저장 + 글쓰기 첨부 택1의 확장.
8. **Dividend Journey** — 연도별 월배당 성장 타임라인, 공유·좋아요·댓글. 커뮤니티+프로필 연결.
9. **투자 배지 시스템** — Gamification, 프로필 표시. 실제 배당 기록(#6)·목표(#1)에 의존.
10. **AI 포트폴리오 분석** — 배당 특화 LLM 리포트(안정성/성장/분산/집중도/FIRE 적합도 점수). Claude API 연동.

- **SNS 공유 카드**(#1)는 동적 OG(`api/og.tsx`) 확장으로 대부분 커버 가능.
