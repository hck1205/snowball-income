# Dividend Reinvestment Simulator — Product Roadmap (v1)

> **비전**
> 단순한 배당 계산기가 아니라, **배당 투자자들이 매일 방문하고 공유하는 플랫폼**을 만든다.
>
> **성장 축**: Calculator → Portfolio → Community → SEO → SNS → Growth

> **핵심 철학**
> - 계산만 하는 서비스는 계산 후 **떠난다**.
> - 기록이 있는 서비스는 **다시 방문한다**.
> - 공유가 있는 서비스는 **사람을 데려온다**.
> - SEO가 있는 서비스는 **사용자를 모은다**.
> - 커뮤니티가 있는 서비스는 **스스로 성장한다**.

*최종 갱신: 2026-07-17*

## 1. 목표 달성 시스템 ★★★★★

**목적** — 사용자가 투자 목표를 만들고 진행 상황을 지속적으로 확인하게 한다.

**설정 항목**
- 목표 월배당 / 목표 연배당 / 목표 FIRE / 목표 은퇴 시점

**자동 계산** — 현재 예상 월배당, 목표 달성률, 이번 달 증가량, 지난달 대비 증가율, 예상 목표 달성일

```
🎯 목표 월배당   300만원
   현재          92만원
   달성률        31%
   예상 달성     2034년 8월
```

**SNS 공유 카드** (자동 생성) — 커뮤니티 / 카카오 / X / Threads
```
🎯 월배당 목표  300만원
   현재         92만원  (31%)
   이번 달      +32,000원
```

## 2. ETF 비교 ★★★★★

**목적** — 투자 전 비교를 서비스 안에서 끝낸다. *(SEO 유입이 매우 강한 기능)*

`SCHD vs VIG` · `SCHD vs JEPI` · `SCHD vs DGRO`

**비교 항목** — 배당률 · CAGR · 총수익 · 배당 성장률 · 배당 지급월 · 운용보수 · 최대낙폭 · 섹터 비중 · 배당 안정성

---

## 3. FIRE Dashboard ★★★★★

**목적** — 기존 FIRE 계산기와 차별화. **현재 포트폴리오를 그대로 이용**한다.

**입력** — 목표 생활비 · 은퇴 목표 · 국민연금 · 기타 수입

```
생활비           300만원
현재 세후 월배당  93만원
생활비 커버율     31%
예상 FIRE        2035년
```

**생활비 항목별 커버 시각화** — 식비 ✅ / 관리비 ✅ / 통신비 ✅ / 주거비 61% / 생활비 31%
→ 생활비를 얼마나 배당으로 해결했는지 시각적으로 보여준다.

## 4. 배당 캘린더 ★★★★★

**목적** — 매달 방문하게 만드는 기능.

```
7월  SCHD · VIG · JEPI · MAIN · O
이번 달 예상 입금  $182
```

**표시 항목** — 배당락일 · 지급일 · 예상 배당금 · 이번 달 총 입금액

---

## 5. ETF 상세 페이지 ★★★★☆

**목적** — SEO 확보. 티커마다 상세 페이지 생성 (`/etf/SCHD`).

**내용** — 배당률 · 운용보수 · 섹터 · 배당 성장률 · 지급월 · 배당 히스토리 · 최근 배당 · 배당 캘린더 · 유사 ETF

---

## 6. 실제 배당 기록 ★★★★☆

**목적** — 예상 배당이 아닌 **실제 받은 배당** 기록.

```
7월  SCHD $54 · JEPI $31
```

**자동 생성** — 월별 그래프 · 연도별 그래프 · 누적 배당 · 목표 대비 진행률

---

## 7. 포트폴리오 복사 + 내 조건으로 시뮬레이션 ★★★★☆

**목적** — 커뮤니티와 계산기를 연결. **서비스의 핵심 바이럴 포인트.**

```
사용자 A: SCHD 50% · VIG 30% · JEPQ 20%
   ↓ [내 조건으로 계산]
사용자 B: 월 투자금 650만원
   ↓
8년 후 월배당 278만원  →  내 포트폴리오 저장
```

---

## 8. Dividend Journey ★★★★☆

**목적** — 투자 성장 기록. 자동 타임라인 생성.

```
2026  12만원 → 2027  31만원 → 2028  58만원 → 2029  87만원
```
공유 가능 · 좋아요 · 댓글

## 9. 투자 배지 시스템 ★★★★☆

**목적** — Gamification. 배지는 프로필에도 표시.

🥉 첫 배당 · 🥈 월배당 10만원 · 🥇 월배당 50만원 · 💎 월배당 100만원 · 🚀 투자 100일 연속 · 📈 목표 달성 50% · 🏆 FIRE 달성

---

## 10. AI 포트폴리오 분석 ★★★★☆

**목적** — LLM을 활용한 개인 리포트. 단순 GPT가 아니라 **배당 투자 특화 분석**.

```
SCHD 70% · JEPI 30%
   ↓
배당 안정성 A · 배당 성장 B+ · 분산도 B · 미국 집중도 높음 · 예상 리스크 보통 · FIRE 적합도 87점
```

## 장기 로드맵 (Phase)

### Phase 1 — 현재 (대부분 완료)
배당 재투자 시뮬레이터 · 포트폴리오 저장 · 로그인 · 커뮤니티 · 공개 URL · (추가 완료) 클라우드 저장 · 멀티 테마 · 프로필

### Phase 2
목표 달성 시스템 · FIRE Dashboard · ETF 비교 · 배당 캘린더

### Phase 3
ETF 상세 페이지 · 실제 배당 기록 · Dividend Journey · 배지 시스템

### Phase 4
AI 포트폴리오 분석 · 포트폴리오 복사 · 커뮤니티 추천 알고리즘 · 인기 포트폴리오 · 인기 투자자

## 구현 참고 — 기존 코드 연결점 & 의존성

> 에이전트/개발 관점의 상세 연결점·병목은 [.claude/knowledge/roadmap.md](../.claude/knowledge/roadmap.md) 참조.

- **이미 있는 기반을 재사용**: 목표(#1)·포트폴리오 복사(#7)·Journey(#8)는 `sim_summary`, 클라우드 저장, 커뮤니티 "시뮬레이터로 열기", 동적 OG 카드(`api/og.tsx`) 위에 얹힌다.
- **⚠️ 티커 데이터 확장이 Phase 2~3 병목**: ETF 비교(#2)·상세(#5)·캘린더(#4)는 배당 히스토리·운용보수·섹터·최대낙폭·지급 스케줄을 요구한다. 현재 티커 데이터(가격·배당률·주기·기대수익률)에 없어, 신규 데이터 소스(FMP 유료 티어 등) 확보가 선행 과제.
- **⚠️ 시계열 스냅샷 공통 모델**: 목표의 "이번 달 증가/지난달 대비"(#1)·실제 배당 기록(#6)·Journey(#8)는 시점별 저장이 필요하다. 한 번 설계하면 셋이 공유 → 먼저 설계할 가치가 있다.
