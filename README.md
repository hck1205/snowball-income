# Yield Architect

`.cursor/rules` 기준으로 구성된 배당 전략 시뮬레이터 MVP입니다.

## Tech Stack

- Vite
- React + TypeScript (strict)
- React Router
- Emotion
- Jotai
- Vitest + React Testing Library

## Run

```bash
npm install
npm run dev
```

## Test / Build

```bash
npm run test
npm run build
```

## Folder Structure

- `components/` 재사용 UI 컴포넌트
- `features/` 기능 단위 모듈
- `pages/` 라우팅 페이지
- `public/` 정적 리소스
- `shared/` 공용 코드

모든 대상 폴더는 `index.ts`를 포함합니다.
