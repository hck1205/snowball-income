Snowball Income — 배당 재투자 시뮬레이터

배당주·ETF로 포트폴리오를 구성하고, 받은 배당을 다시 투자했을 때 자산과 월 배당이 어떻게 불어나는지 시뮬레이션하는 무료 웹 계산기입니다. 초기 투자금·월 투자금·배당률·배당 성장률·투자 기간·세율을 입력하면 연도별 자산 추이, 세후 배당 현금흐름, 목표 월 배당 도달 시점을 계산해 보여줍니다.

주요 기능

  • 배당 재투자(스노우볼) 효과 시뮬레이션 — 당월/익월 재투자 시점 선택
  • 여러 종목의 포트폴리오 비중 조절 및 특정 종목 비율 고정
  • 목표 월 배당 도달 시점 계산
  • 연도별 자산·배당 추이와 월별 실지급 배당 확인
  • 배당소득세와 전량 매도 시 예상 양도소득세 추정
  • 시나리오 탭으로 여러 전략을 나란히 비교, 링크로 공유

사용 방법

  1. 추천 포트폴리오를 고르거나 티커를 직접 만들어 종목을 추가합니다.
  2. 포트폴리오 구성에서 종목별 비중을 조절합니다.
  3. 초기 투자금·월 투자금·투자 기간·세율 등 투자 설정을 입력합니다.
  4. 최종 자산, 월 배당, 목표 도달 시점 등 시뮬레이션 결과를 확인합니다.

이 앱은 입력한 가정을 그대로 계산해 보여주는 시뮬레이터입니다. 투자 자문이 아니며, 참고용으로만 사용해 주세요. 실제 수익률과 배당은 시장 상황에 따라 달라집니다.
